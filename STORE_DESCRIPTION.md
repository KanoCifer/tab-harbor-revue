# Tab Harbor reVue - Chrome Web Store 描述

智能标签页管理器，自动分组、快速搜索、待办事项和快捷链接，让您的浏览器整洁有序。

Smart tab manager with auto-grouping, quick search, todos, and quick links. Keep your browser organized and productive.

Tab Harbor reVue 将您的新标签页转变为高效的工作空间。它帮助您管理混乱的标签页、跟踪任务并快速访问常用网站。

主要用途：

• 自动按域名分组标签页 - 不再需要手动整理数十个标签，扩展会自动将它们分类，让您一目了然地找到所需内容

• 实时搜索过滤 - 在搜索框中输入关键词即可快速定位标签页，按 Enter 键可直接使用默认搜索引擎搜索

• 内置待办事项 - 无需打开额外的任务管理应用，直接在浏览器中创建、排序和完成任务，支持拖拽调整优先级

• 快捷链接 - 将常用网站固定在起始页，一键访问，提高工作效率

• 稍后阅读功能 - 遇到重要但暂时没时间看的网页，保存到延迟列表，避免关闭后丢失

• 深色/浅色主题 - 根据您的喜好切换，保护视力

为什么应该安装：

✓ 节省时间 - 快速找到和管理标签页，减少查找时间
✓ 提高效率 - 集成任务管理，无需切换应用
✓ 保持专注 - 整洁的界面帮助您专注于当前任务
✓ 隐私安全 - 所有数据本地存储，不上传云端，不收集个人信息
✓ 完全免费 - 开源软件，无广告，无订阅费用

适合人群：任何希望保持浏览器整洁有序的用户。

Tab Harbor reVue transforms your new tab page into a productive workspace. It helps you manage chaotic tabs, track tasks, and quickly access favorite
websites.

Key Benefits:

• Auto-group tabs by domain - No more manually organizing dozens of tabs. The extension automatically categorizes them, making it easy to find what
you need at a glance

• Real-time search and filter - Type keywords in the search box to quickly locate tabs. Press Enter to search directly with your default search engine

• Built-in todo list - Create, sort, and complete tasks directly in your browser without opening separate task management apps. Drag and drop to
adjust priorities

• Quick links - Pin frequently visited websites to your start page for one-click access and improved productivity

• Save for later - Temporarily save important webpages you don't have time to read now, preventing loss when closing tabs

• Dark/Light themes - Automatically switch based on your preference or system settings to reduce eye strain

Why Install:

✓ Save time - Quickly find and manage tabs, reducing search time
✓ Boost productivity - Integrated task management without switching apps
✓ Stay focused - Clean interface helps you concentrate on current tasks
✓ Privacy protected - All data stored locally, never uploaded to cloud, no personal information collected
✓ Completely free - Open source software, no ads, no subscription fees

Perfect for anyone who wants to keep their browser organized and efficient.
